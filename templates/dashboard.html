XP Formula

Each message earns 5–15 XP randomly (capped at 1 per 60s)

Level formula: level = floor((xp / 50) ^ 0.5)

Level thresholds: Lv1=50, Lv2=200, Lv5=1250, Lv10=5000, Lv25=31250

Danger Zone
Irreversible actions. All require confirmation. These affect the live bot immediately.
Emergency Lockdown
Locks every text channel in the server instantly. Equivalent to running /lockdown in Discord. Use during raids or raids.
Lift Lockdown
Restores send permissions for everyone in all channels. Equivalent to /unlockdown.
Reset Bot Config
Wipes all channel and role configuration for this server. You will need to re-run setup. Does not affect XP, economy or member data.
Shut Down Bot
Gracefully shuts down the bot process. Requires the DASHBOARD_SECRET env variable to match. Bot will need to be restarted manually.
